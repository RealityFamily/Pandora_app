Для того чтобы создать установщик необходимо использовать утилиту Inno Setup Compiler.

В папке приложен скрипт который открывается в данной утилите.

Для того чтобы .Net 5.0 установился сам без помощи юзера интегрирован скрипт который использует Inno Download Plugin под капотом. Этот плагин и позволяет выкачивать нужную версию .Net.

Всю статью на базе которой был написан скрипт автоинсталяции можно найти [!тут](https://engy.us/blog/2021/02/)
